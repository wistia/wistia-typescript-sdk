# PostFoldersPublicEnum

A flag indicating whether or not the folder is enabled for public access. Set to “1” to enable and “0” to disable.

## Example Usage

```typescript
import { PostFoldersPublicEnum } from "@wistia/wistia-api-client/models/operations";

let value: PostFoldersPublicEnum = "1";
```

## Values

```typescript
"0" | "1"
```
