# PostTagsResponse

A tag is used to tag related media. You can then filter media
by a specific tag.


## Example Usage

```typescript
import { PostTagsResponse } from "@wistia/wistia-api-client/models/operations";

let value: PostTagsResponse = {
  name: "My tag Title",
  taggingsCount: 2,
  created: new Date("2010-08-13T18:47:39+00:00"),
  updated: new Date("2010-08-19T21:47:00+00:00"),
};
```
