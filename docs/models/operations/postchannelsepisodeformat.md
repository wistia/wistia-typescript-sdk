# PostChannelsEpisodeFormat

The format for episodes for the podcast channel. This parameter only takes effect if podcasting is enabled for the channel.

## Example Usage

```typescript
import { PostChannelsEpisodeFormat } from "@wistia/wistia-api-client/models/operations";

let value: PostChannelsEpisodeFormat = "episodic_with_seasons";
```
