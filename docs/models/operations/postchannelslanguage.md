# PostChannelsLanguage

The ISO 639-1 language code for the channel. This parameter only takes effect if podcasting is enabled for the channel.

## Example Usage

```typescript
import { PostChannelsLanguage } from "@wistia/wistia-api-client/models/operations";

let value: PostChannelsLanguage = "sl";
```
