# MediaShowType

A string representing what type of media this is.

## Example Usage

```typescript
import { MediaShowType } from "wistia/models";

let value: MediaShowType = "Video";
```

## Values

```typescript
"Video" | "Audio" | "Image" | "PdfDocument" | "MicrosoftOfficeDocument" | "Swf" | "UnknownType"
```
